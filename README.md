 Chest X-Ray Pneumonia Detection Using CNN

 About the Project

This is my first deep learning project using TensorFlow and Keras.

The goal of this project is to classify chest X-ray images into two categories:

* NORMAL
* PNEUMONIA

I built this project to learn how Convolutional Neural Networks (CNNs) work for image classification.

Dataset

This project uses the Chest X-Ray Pneumonia dataset.

The dataset contains chest X-ray images divided into:

* train
* test
* val

Each folder contains images from two classes:

* NORMAL
* PNEUMONIA

Project Workflow

The project follows these steps:

1. Load the chest X-ray images.
2. Resize all images to 100 ? 100 pixels.
3. Normalize the pixel values.
4. Convert the images into NumPy arrays.
5. Split the dataset into training and testing sets.
6. Build a CNN model.
7. Train the model.
8. Evaluate the model.
9. Save the trained model.
10. Predict a random test image.

CNN Model

The model contains:

* Conv2D
* MaxPooling2D
* Flatten
* Dense (128 neurons)
* Dense (1 neuron with Sigmoid activation)


 Libraries Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Scikit-learn
* Pillow


 Model Evaluation

After training, I evaluated the model using:

* Accuracy
* Confusion Matrix
* Classification Report
* ROC Curve
* AUC Score

My model achieved approximately 98% accuracy on the test dataset.
